# MalditosTests
repositorio de tests gamberros

https://vcastellar.github.io/MalditosTests/

## Uso rápido

Script para crear la estructura base de un test:

```bash
./scripts/create-test.sh mi_test
```

Esto crea:

- `tests/mi_test/index.html`
- `assets/data/mi_test.js`

Si no pasas el nombre del test, el script muestra el uso y termina con error.
